#Projet l'abime

##/characters
-contient une fiche résumé pour l'ensemble des personnages qui apparaîtront dans le jeu
-protagonist.md contient les informations sur le personnage principal que va incarner le joueur
-/npc non playable character contient les personnages non jouables par le joueur
-/pc playable character contient les personnages jouables par le joueur

##/evaluation
-contient les différents systèmes d'évaluation attendu en fonction du type de dialogues

##/global
-contient des informations et des instructions valables pour tous les dialogues qui auront lieu dans le jeu
-dialogue_style_dialogue le style des dialogues attendu dans le jeu
-player_experience
-world_rules les règles du monde que le construit, le lore 

##/dialogues
-contient les scènarios et les dialogues pour chaque scène de chaque mission
-/scènes un fichier.md qui décrit le scénario et les différentes versions en .ink de ce dialogue

#/style
-contient le style attendu pour chaque dialogue
